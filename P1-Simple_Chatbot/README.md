# 🤖 Simple Chatbot

A beginner-friendly rule-based chatbot developed using Python. The chatbot interacts with users by responding to predefined questions and messages using conditional statements.

---

## 📌 Project Overview

This project demonstrates the fundamentals of chatbot development using Python. It accepts user input and provides predefined responses based on the entered message.

---

## 🎯 Objective

- Build a simple chatbot using Python
- Understand user input handling
- Implement decision-making using conditional statements
- Create an interactive console-based application

---

## 🚀 Features

- 💬 Interactive conversation
- 🤖 Predefined responses
- 📝 Accepts user input
- 🔄 Continuous chat until the user exits
- 😊 Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python

## ▶️ How to Run

### 1️⃣ Install Dependencies

```bash
uv sync
```

### 2️⃣ Activate Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

### 3️⃣ Run the Project

```bash
python main.py
```
