# 🌸 Data Classification Using AI

A simple Machine Learning project that classifies Iris flowers into different species using the **Decision Tree Classification** algorithm.

---

## 📌 Project Objective

The objective of this project is to build a basic classification model using the Iris dataset. The model learns from the training data and predicts the flower species based on sepal and petal measurements.

---

## 🚀 Features

- 📂 Loads the Iris dataset
- 📊 Splits the dataset into training and testing sets
- 🤖 Trains a Decision Tree Classification model
- 🌸 Predicts the flower species
- 📈 Displays model accuracy
- 👨‍💻 Accepts user input for prediction

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Decision Tree Classifier

## 📚 Dataset

This project uses the **Iris Dataset**, which is available in the Scikit-learn library.

The dataset contains:

- 150 flower samples
- 3 flower species
  - Setosa
  - Versicolor
  - Virginica
- 4 features
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
