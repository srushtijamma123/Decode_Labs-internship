# 🎬 AI Movie Recommendation System

A simple AI-based Movie Recommendation System built using Python and Streamlit.

The application recommends movies based on the user's selected Genre, Language, and Mood.

---

## 🚀 Features

- Interactive Streamlit UI
- Select Movie Genre
- Select Language
- Select Mood
- Recommends matching movies
- Displays movie details:
  - Movie Name
  - Rating
  - Genre
  - Language
  - Mood
  - Short Description
- Displays a friendly message if no matching movies are found

---

### ▶️ How to Run

Clone the repository
```
git clone <repository-url>
```
Move into the project folder

```
cd P3-Movie_Recommendation_System
```

Create virtual environment

```
uv sync
```

Activate virtual environment

Windows

```
.venv\Scripts\activate
```

Run the application

```
streamlit run app.py
```
